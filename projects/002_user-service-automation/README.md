Project 002 - User & Service Automation

Overview
This project automates common Linux administration tasks related to user account management and service administration across multiple Linux servers.

The solution helps administrators create, modify, disable, and audit user accounts while also managing system services remotely from a centralized Rocky Linux controller.

Objectives
• Automate Linux user administration tasks.
• Create and manage user accounts across multiple servers.
• Automate service administration and monitoring.
• Standardize administrative operations.
• Reduce manual effort and configuration errors.
• Improve operational efficiency.
