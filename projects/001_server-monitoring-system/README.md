# Linux Monitoring Toolkit

## Overview

This project provides Bash scripts to monitor Linux system performance
on multiples servers.

It helps administrators quickly check system health including CPU,
memory, disk usage.


## Features

- CPU usage monitoring
- Disk space monitoring
- Memory usage monitoring


## Technologies Used

- Bash
- Linux system tools (top, free, df)


## What I Learned

- Bash scripting
- Linux system monitoring
- Troubleshooting performance issues
