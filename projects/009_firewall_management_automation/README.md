Project 009 - Firewall Management Automation
Overview

This project automates Linux firewall administration across multiple servers using Bash, Python, and Ansible.

The solution manages firewall rules, validates network access, audits open ports, and standardizes firewall configurations across managed Linux systems. It demonstrates real-world infrastructure security practices commonly performed by Linux Administrators, Systems Administrators, and Infrastructure Engineers.

Objectives
• Automate firewall rule management across multiple Linux servers.
• Standardize firewall configurations.
• Validate allowed network services.
• Audit open ports and firewall rules.
• Improve infrastructure security.
• Reduce manual firewall administration.
• Generate firewall compliance reports.
