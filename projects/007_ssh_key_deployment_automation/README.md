Project 007 - SSH Key Deployment Automation
This project automates the deployment of SSH public keys across multiple Linux servers using Ansible.

Objectives
•	Automate SSH public key deployment.
•	Enable passwordless authentication for remote administration.
•	Reduce manual configuration effort.
•	Improve consistency across managed hosts.
•	Support future Ansible automation projects.
