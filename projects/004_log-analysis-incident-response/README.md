 Project 004 - Log Analysis & Incident Response Automation

## Overview

This project automates log collection, analysis, and incident response tasks across multiple Linux servers.

The solution helps administrators identify system errors, authentication failures, service issues, and suspicious activity by collecting and analyzing log data from managed Linux servers.

The project also demonstrates how automation can improve troubleshooting, operational visibility, and incident response workflows.

---

## Objectives
• Automate Linux log collection.
• Analyze authentication and system logs.
• Detect failed login attempts.
• Identify service-related errors.
• Generate centralized incident reports.
• Improve troubleshooting and incident response capabilities.
• Reduce manual investigation effort.
