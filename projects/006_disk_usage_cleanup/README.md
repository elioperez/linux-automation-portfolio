Project 006 - Disk Usage Cleanup Automation

## Overview

This project automates disk usage monitoring and cleanup operations across multiple Linux servers.

The solution helps administrators identify storage consumption issues, locate large files and directories, remove unnecessary data, and maintain healthy disk utilization levels. It demonstrates proactive system administration practices used to prevent outages caused by insufficient disk space.

---

## Objectives
• Automate disk usage monitoring across multiple Linux servers.
• Identify large files and directories.
• Detect storage utilization issues.
• Automate cleanup operations.
• Generate disk usage reports.
• Improve system availability and performance.
• Reduce manual maintenance effort.
