This project automates the detection and cleanup of disk space consumption across multiple Linux servers.

The purpose of this project is to automatically identify disk usage issues, locate large files and directories, generate usage reports, and perform cleanup operations using Bash, Python, and Ansible.

Objectives
Monitor disk utilization across multiple Linux systems.
Identify filesystems with high disk usage.
Locate large files and directories consuming storage.
Generate disk usage reports.
Automate cleanup operations.
Improve system availability and performance.
Demonstrate Linux administration and automation skills.

Linux (Rocky Linux)
Bash
Python
